# E-Commerce Web Application Testing Project

## Overview

This project demonstrates manual and basic automation testing of an
e-commerce web application using the SauceDemo demo site.

## Modules Covered

-   Login
-   Product Listing
-   Product Details
-   Cart
-   Logout

## Testing Types

-   Functional Testing
-   Smoke Testing
-   Regression Testing

## Tools & Technologies

-   Selenium WebDriver
-   Java
-   TestNG
-   Jira (simulated)
-   Excel for Test Cases and Bug Tracking

## Deliverables

-   Test Scenarios (15)
-   Test Cases (25)
-   Bug Reports (8)
-   Sample Selenium automation scripts
